Simple Shell [MODEL_666]

A basic command line shell written in C that can execute system commands

Getting Started [Open the file where saved in terminal]

1. Compile code
	
	gcc -o model_666 model_666.c
2. Run

	./model_666

What It Can Do

- Run basic system commands like ls, pwd, echo
- Change directories with cd
- Exit the shell with exit

Example Usage

	model_666> ls
	model_666> cd Documents
	model_666> pwd
	model_666> exit

Requirements

- GCC compiler
- Linux or macOS
